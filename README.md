Sistema de Clientes (CLI)
=========================

Descrição
---------
Sistema simples em Python para gerenciamento de clientes via terminal.
Permite cadastrar, listar, editar e excluir clientes, com persistência de dados em arquivo JSON.

Objetivo do projeto
-------------------
Este projeto foi desenvolvido com o objetivo de praticar lógica de programação, organização de código e controle de versão utilizando Git.

Funcionalidades
---------------
- Cadastro de clientes
- Listagem de clientes
- Edição de clientes
- Exclusão de clientes
- Persistência de dados com JSON

Tecnologias utilizadas
----------------------
- Python
- JSON
- Git

Como executar
-------------
1. Clone o repositório:
   git clone https://github.com/nasu-dev/sistema-clientes.git

2. Acesse a pasta do projeto:
   cd sistema-clientes

3. Execute o arquivo:
   python main.py

Observações
-----------
- O arquivo "clientes.json" será criado automaticamente na primeira execução.
- Não é necessário criar o arquivo manualmente.
